# Banking

from: REST based microservices API development in Go

[original sources](https://github.com/ashishjuyal/banking)

init module

```
go mod init github.com/rastislavsvoboda/banking
```

build

```
go build
```

run without build

```
go run .\main.go
```

or after build

```
banking.exe
```
